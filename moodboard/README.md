# Ambient Moodboard

A React-based ambient moodboard application built with Vite.

## Requirements

- Node.js >= 16
- npm or yarn package manager

## Quick Start

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm run preview
```

For static preview of the build output, you can use http-server:
```bash
npx http-server dist
```
